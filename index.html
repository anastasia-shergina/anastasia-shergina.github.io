Утилізація

PET(E) або ПЕТ

В нього запаковані: вода, лимонад, соки; молоко, кефір, йогурт та інші кисломолочні напої; рослинна олія; кетчуп та інші соуси; спеції та сипучі продукти; косметика порошки.

Утилізація

PEHD (HDPE) або ПЕВП

З нього роблять: шарудливі пакетики,пляшки для соку, молока, йогурта і кришки; одноразовий посуд; контейнери для їжі; флаконы для косметики; банки для побутової хімії; каністри; відра і тази; дитячі іграшки.

Утилізація

PVC або ПВХ

З нього роблять: контейнери для їжі та харчову плівку; пластикові вікна, натяжні стелі, деталі для меблів, труби; скатертини; лінолеум та штучну шкіру; тарудля технічних рідин дитячі іграшки.

Утилізація

LDPE або PEBD

З нього роблять: майже всі пакети; харчову плівку; цупкі непрозорі пакети; пакети для мийних засобів та прального порошка; пакети для заморожених продуктів, частину упакування для побутової техніки.

Утилізація

PP

З нього роблять: стаканчики для сметани, йогуртів, морозива упаковку для шоколадок, хліба упаковку для дитячого харчування, підгузки, харчові контейнери, трубочки баночки для таблеток, шприци, дитячі іграшки, лотки для холодильника.

Утилізація

PS

З нього роблять: непрозорий одноразовий посуд; банки для йогуртів; вспінені лотки; коробки для овочів та фруктів; пенопласт; аудіокасети та CD; дитячі іграшки; теплоізоляційні та сендвіч-панелі.

Утилізація

O або OTHER

З нього роблять: пляшки для дітей; прозорий одноразовий посуд; багаторазові пляшки; дитячі іграшки; тару для зубної пасти; пакети для випічки; комбінована упаковка; CD та DVD.

PET(E) або ПЕТ

×
На переробку беруть всюди, де є роздільний збір сміття. Виняток - пляшки з-під олії - масло проникає в пластик і не дає можливості виробляти з нього вторсировину. Використовувати можна тільки один раз.

PEHD (HDPE) або ПЕВП

×
На переробку беруть всюди, де є роздільний збір сміття. Використовувати можна кілька разів.

PVC або ПВХ

×
На переробку не приймають, уникайте. Протипоказаний для харчових продуктів, але все-таки використовується. З іншою метою можна використовувати багато разів.

LDPE або PEBD

×
На переробку приймають, але не всюди. Використовувати можна декілька разів. Найменш шкідливий.

PP

×
На переробку приймають, але не всюди. Використовувати можна декілька разів. Майже нешкідливий.

PS

×
На переробку приймають не всюди. Намагайтесь уникати. Використовувати можна лише один раз.

O або OTHER

×
На переробку не приймають. Уникайте. Використовувати можна декілька разів.
